TROUBLESHOOTING
===============

If imports fail:
  1. Ensure all requirements installed: pip install -r requirements.txt
  2. Check Python version >= 3.8 (uses asyncio.run)
  3. Verify .env file exists with required keys

If agents won't start:
  1. Check event_bus.subscribe calls in agent.run()
  2. Ensure no circular event dependencies
  3. Check logger.info messages for startup trace

If strategies don't generate signals:
  1. Verify OHLCV data has sufficient length (20+ candles)
  2. Check for NaN values in OHLCV data
  3. Review confidence scores (may be 0 if insufficient data)

If executor has issues:
  1. Verify order quantity > 0
  2. Check cash available for buy orders
  3. Review fill_price calculation and slippage
